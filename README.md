## Kibana资源汉化项目

---
**注意：此项目适用于Kibana 5.x-6.x的任意版本，汉化前请注意备份！**

意见反馈：redfree@anbai.com

###使用方法：
```
Python main.py Kibana目录
```
稍等一会一个全新的中文版Kibana将呈现在你面前。
